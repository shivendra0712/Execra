# 🐍 Execra SDK

Python SDK for **Execra** — the Universal Execution Intelligence Layer. Embed Execra's guidance pipeline into your own Python applications programmatically.

## 🚀 Installation

```bash
pip install execra-sdk
```

## 🛠️ Quick Start

```python
import asyncio
from execra_sdk import ExecraClient, ExecraConnectionError

async def main():
    # Initialize the client
    client = ExecraClient()
    
    # Connect to the Execra instance
    client.connect("http://localhost:8000", api_key="your-api-key")

    try:
        # Ask for guidance
        result = await client.ask("How do I fix the IndexError on line 42?")
        print(f"[{result.confidence:.0%}] {result.answer}")
        print(f"Reasoning: {result.reasoning}")
        
    except ExecraConnectionError as e:
        print(f"Could not reach Execra: {e}")
    finally:
        # Always close the client
        await client.close()

if __name__ == "__main__":
    asyncio.run(main())
```

## 📖 Key Features

### 📡 Real-time Streaming
Stream guidance instructions as they are generated by the Execra engine.

```python
async for instruction in client.stream_guidance():
    print(f"New Instruction: {instruction.instruction}")
```

### 📊 System Status
Check the health and configuration of your Execra instance.

```python
status = await client.get_status()
print(f"Status: {status.status}, Version: {status.version}")
```

### 🔄 Mode Switching
Dynamically switch between different execution modes (e.g., `autonomous`, `manual`, `coding`).

```python
response = await client.switch_mode("autonomous")
print(response.message)
```

### 📜 Action History
Retrieve the history of actions taken or guided by Execra.

```python
actions = await client.get_actions(limit=10)
for action in actions:
    print(f"{action.timestamp}: {action.description}")
```

## 🧪 Error Handling

The SDK provides specific exceptions for better error management:

- `ExecraConnectionError`: Raised when the server is unreachable or timing out.
- `ExecraAuthError`: Raised when the API key is invalid or missing (401/403).
- `ExecraError`: Base class for other API-related errors.
